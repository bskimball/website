# Brian Kimball's Personal Website

Personal website and blog built with Astro, React, and Tailwind CSS.

**Live site:** https://brian-kimball.com

## Development

```bash
npm install        # Install dependencies
npm run dev        # Start dev server
npm run build      # Type check + build
npm run preview    # Preview production build
npm run lint       # Check for linting errors
npm run lint:fix   # Auto-fix linting errors
npm run format     # Format code with Prettier
```

## Tech Stack

- Astro 5
- React 19
- Tailwind CSS 4
- shadcn/ui components
- ESLint + Prettier

See [AGENTS.md](./AGENTS.md) for more details about the project structure and conventions.
